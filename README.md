<h2>flask-blog</h2>
<p>一个基于 Flask 的简单博客系统</p>

<h3>使用步骤</h3>
<pre>
<code>
$ pip install -r requirements.txt
#然后在MySQL中手动建立一个名为weblog的数据库
$ python manage.py db upgrade
$ python manage.py runserver 
</code>
</pre>
<br />
<h3>博客功能</br>
<p>
<ul>
<li>注册/登录系统</li>
<li>个人信息设置、头像上传</li>
<li>用户动态展示</li>
<li>站内消息功能</li>
<li>用户关注系统/话题关注系统</li>
<li>博客包含技术问答板块和主题文章板块</li>
<li>文章评论</li>
<li>问答系统</li>
<li>文章标题和内容支持关键字搜索</li>
<li>简单的后台管理系统（完成中...）</li>
<li>......</li>
